## Роуты и переходы между страницами

| Страница | Роут | Описание |
|----------|------|----------|
| Страница рабочего пространства | `/workspace/{ws_id}/dashboard` | ID выбранного в данный момент пространства |
| Страница проекта | `/workspace/{ws_id}/project/{p_id}` | Проект `{p_id}` в пространстве `{ws_id}` |
| Команда | `/workspace/{ws_id}/project/{p_id}/members` | Управление командой |
| Автоматическая разметка | `/workspace/{ws_id}/project/{p_id}/auto` | Настройка и запуск авторазметки |
| Экспорт | `/workspace/{ws_id}/project/{p_id}/export` | Выбор формата и скачивание |
| Настройки | `/workspace/{ws_id}/project/{p_id}/settings` | Изменение параметров проекта |
| Ручная разметка (открывается при нажатии на изображение) | `/workspace/{ws_id}/project/{p_id}/editor/{img_id}` | Экран ручной разметки изображения |

### Переход 1: Смена пространства
- **Действие:** Пользователь нажимает на стрелочку «Мое рабочее пространство» и выбирает «Рабочее пространство №2».
- **Результат:** Роут меняется с `/workspace/1/dashboard` на `/workspace/2/dashboard`. Список проектов обновляется.

### Переход 2: Открытие страницы проекта
- **Действие:** Пользователь нажимает «Открыть» на карточке проекта (например, «Мой проект2»).
- **Результат:** Роут меняется с `/workspace/1/dashboard` на `/workspace/1/project/2`. Открывается страница проекта.
- На странице проекта при нажатии на кнопки «Команда», «Автоматическая разметка», «Настройки», «Экспорт» открываются соответствующие страницы.
- При нажатии на изображение из галереи открывается редактор ручной разметки.

## Источники данных

### Экран 1
| Элемент | Метод | Endpoint | Описание данных |
|---------|-------|----------|-----------------|
| Выбор рабочего пространства | GET | `/api/v1/workspaces` | Массив доступных пространств пользователя |
| Список карточек проектов | GET | `/api/v1/workspaces/{ws_id}/projects` | Название, тип задачи, общее количество изображений, количество размеченных изображений |
| Статус по лимитам | GET | `/api/v1/workspaces/{ws_id}/usage` | Объект с названием тарифа, текущим и максимальным использованием для проектов и изображений |
| Новый проект | POST | `/api/v1/workspaces/{ws_id}/projects` | Отправка данных: название, описание, тип, список
