# Коды ошибок API (шаблон)

| HTTP-код | Описание |
|----------|----------|
| 400 | Неверный запрос (например, отсутствует обязательное поле) |
| 401 | Неавторизован |
| 403 | Доступ запрещён |
| 404 | Ресурс не найден |
| 413 | Слишком большой файл |
| 422 | Ошибка валидации данных |
| 429 | Превышен лимит запросов |
| 500 | Внутренняя ошибка сервера |

В теле ответа всегда возвращается JSON с полями `error` (string) и `code` (string).
