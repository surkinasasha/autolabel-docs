# AutoLabel — умная разметка изображений

Веб-платформа для разметки изображений с использованием open‑source моделей (Grounding DINO, SAM) и удобным интерфейсом коррекции.

**Ключевые возможности:** автоматическая детекция/сегментация по текстовому промпту, сегментация по клику, пакетная обработка, экспорт в COCO / YOLO / Pascal VOC.
