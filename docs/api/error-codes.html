Коды ошибок API (шаблон)

HTTP-кодОписание
400Неверный запрос (например, отсутствует обязательное поле)
401Неавторизован
403Доступ запрещён
404Ресурс не найден
413Слишком большой файл
422Ошибка валидации данных
429Превышен лимит запросов
500Внутренняя ошибка сервера

В теле ответа всегда возвращается JSON с полями error (string) и code (string).
